Bandwidth & Throughput Calculations

1.0 Throughput For A Single Instance of A Protocol

1.1 Throughput Calculation

The formula for throughput is given by:

\[ \text{throughput} = \frac{(\text{totalBytesSent} + \text{sizeof(PacketHeader)} \times \text{sequenceNumber}) \times 8}{\text{duration}} \]

where:

  • totalBytesSent is the total number of bytes of the payload sent.
  • sizeof(PacketHeader) is the size of the header of each packet.
  • sequenceNumber is the total number of packets sent.
  • 8 converts bytes to bits.
  • duration is the time taken for the transmission in seconds.

2.0 Bandwidth For A Single Instance of A Protocol

2.1 Bandwidth Calculation

The formula for bandwidth is given by:

\[ \text{bandwidth} = (\frac{\text{throughput}}{\text{LINK_CAPACITY}}) \times 100 \]

where:

  • throughput is bits sent per second
  • LINK_CAPACITY is the maximum amount of data that can be transmitted over a communication channel within a given time frame.
  • 100 converts the value to a bandwidth percentage.
