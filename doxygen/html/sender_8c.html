Detailed Description

Functions for sending data over an enhanced UDP protocol.

This file contains the implementation of functions responsible for sending data over the enhanced UDP protocol. It includes a key function rsend which handles sending packets and handling acknowledgments.

Entry point for the UDP file sender program.

This function parses command line arguments and initiates the file transfer process by calling the rsend function. The program expects exactly four arguments: the receiver's hostname, the receiver's UDP port, the filename to transfer, and the number of bytes to transfer.
