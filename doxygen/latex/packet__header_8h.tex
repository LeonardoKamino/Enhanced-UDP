\doxysubsection{Detailed Description}
Definitions and utility functions for UDP protocol flag handling. 

This header file defines constants for packet flags and includes utility functions for setting and checking these flags within the \doxylink{struct_packet_header}{Packet\+Header} structure. These are used to manage and interpret the state of packets in a custom UDP protocol.
